This is LaTeX template for a CV(Curriculum Vitae), Résumé or Cover Letter 

#### Requirements
1. You need to install docker and run it
2. For Windows PowerShell, run: `docker run --rm -v ${PWD}:/work csmith/awesome-cv-builder`
3. For Linux/Mac, run: `docker run -u $UID:$GID --rm -v $PWD:/work csmith/awesome-cv-builder`
4. Then you can see the output: cv.pdf
